列表

  • dl标签

    dt：定义列表中的项目

    dd：描述列表中的项目

表格

表格 描述
< table > 定义表格
< caption > 定义表格标题。
< th > 定义表格的表头。
< tr > 定义表格的行。
< td > 定义表格单元。
< thead > 定义表格的页眉。
< tbody > 定义表格的主体。
< tfoot > 定义表格的页脚。

iframe框架

属性 值 描述
width px/% 规定 iframe 的宽度。
height px/% 规定 iframe 的高度。
frameborder 1/0 规定是否显示框架周围的边框。
scrolling yes/no/auto 规定是否在 iframe 中显示滚动条。
src URL 规定在 iframe 中显示的文档的 URL。
seamless seamless 规定在 < iframe > 中显示的页面的 HTML 内容。
